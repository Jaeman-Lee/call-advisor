# 로컬·원격 Whisper 모델 평가

## 평가 세트

폰에 있던 개인 녹음에서 짧은 파일 4개 전체와 긴 파일 3개의 시작·중간·끝 30초를
추출했습니다. 총 13개, 354.03초입니다. 음성, 원본 경로와 전사 원문은 Git 및
GitHub Release에 포함하지 않습니다.

모든 PC 시험은 Ryzen 5 5600X, WSL2, CPU-only, 6스레드, 동일 Silero VAD 조건입니다.

## 결과

|---|---:|---:|---:|---:|---:|

| medium Q5_0 | 319.28초 | 24.56초 | 40.18초 | 1100.4MB | 1132.1MB |

- medium Q5_0 명확히 우세: 10개
- 무음으로 동률: 1개
- 두 모델 모두 판단 곤란: 2개
- base가 명확히 우세: 0개

정답을 확정할 수 있는 52자 한국어 시험 샘플에서는 base CER 34.6%, medium Q5_0
CER 26.9%로 medium Q5_0의 글자 오류가 상대적으로 약 22% 감소했습니다.

base는 일부 어려운 구간에서 같은 문장을 반복 생성했습니다. medium Q5_0은 숫자,
업무 용어와 긴 문맥을 더 안정적으로 복원했지만, 매우 짧거나 비음성에 가까운 입력은
두 모델 모두 신뢰하기 어려웠습니다.

## 운영 결정

```text
PC 정상  → medium Q5_0
PC 장애  → 스마트폰 base
```

medium Q5_0은 base보다 약 4.5배 느리고 메모리를 약 3.8배 사용하지만 전체 평가에서는
평균 1.11배속으로 처리되어 통화 후 배치 목적을 충족했습니다. 실시간 요구가 없고
정확도를 우선하는 현재 용도에서는 이 구성이 적합합니다.
